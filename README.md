# Corne alike keyboard
Split keyboard with 3x6 column staggered keys, 3 keys thumb cluster, Corne inspired

## Features

- Wireless only (no TRRS connectors)
- JST battery connectors
- On/Off switch
- Kailh Choc Switch v1 spacing
- No OLED module
- No Backlighting
